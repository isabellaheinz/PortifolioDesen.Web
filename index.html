Desenvolvimento para Web

A Disciplina Desenvolvimento para Web, é uma das materias integradas ao 2° ano do Ensino Médio no IFPR campus Foz do Iguaçu.

Durante a disciplina, Além de compreender os fundamentos da organização e estruturação de uma página webos Estudantes estarão aptos a criar sites para web utilizando HTML, CSS e JavaScript.

Como funciona o Html?

  • Todas os sites e aplicações web do mundo usam HTML!
  • Linguagem exclusivamente para o desenvolvimento de Interfaces;
  • Tags são organizadas por categorias:
    • Metadados;
    • Estruturais;
    • de layout;
    • de texto;
    • de mídia;
    • formulários;
  • O HTML é uma linguagem de marcação, não de programação: não é possível executar algoritmos com HTML -> apenas criar telas.

Como funciona o CSS?

  • CSS é a linguagem usada para definir o estilo e o layout de páginas da web;
  • Com o CSS, você pode controlar a aparência de elementos HTML, como cores, fontes, espaçamentos, tamanhos e posicionamento:
    • Estilização: Define cores, fontes, tamanhos e mais;
    • Layout: Organiza elementos na tela (ex.: Flexbox e Grid);
    • Responsividade: Adapta a página para diferentes dispositivos;
    • É necessário selecionar os elementos HTML.
